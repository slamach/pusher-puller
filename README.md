# Pusher & Puller

## Getting started
```
npm install
```

To enable pre-commit hook you need to run `npm install` already in git repository.

If you initialize git repostitory after installing, additionally run `npm run prepare`.

### Development
```
npm start
```

### Production
```
npm build
```
