## About the Project

Web 3.0 project created for learning purposes.

You can authorize with Metamask wallet, send ETH, attach messages and images from [Giphy](https://giphy.com). Deployed project works over Goerli Test Network.

- React
- Redux
- Ethers

- Hardhat
